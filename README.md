
# מערכת ניהול AllForYou 🚀

מערכת ניהול מתקדמת לעסקים הבנויה בטכנולוגיות מודרניות עם תמיכה מלאה בעברית (RTL).

## 📋 אודות המערכת

מערכת AllForYou היא פלטפורמה מקיפה לניהול עסקים הכוללת:
- ניהול עובדים ומשמרות
- מערכת טוקנים למילוי זמנים
- ניהול מסמכים וחתימות דיגיטליות
- דוחות נוכחות ושעות עבודה
- אינטגרציות עם מערכות חיצוניות
- מערכת הרשאות מתקדמת

## 🔐 סוגי משתמשים

| סוג משתמש | הרשאות |
|-----------|---------|
| **Super Admin** | גישה מלאה לכל המערכת, ניהול עסקים מרובים |

## 🛠 טכנולוגיות

### Frontend
- **React 18** - ספריית JavaScript מודרנית
- **TypeScript** - טיפוסים סטטיים
- **Tailwind CSS** - עיצוב רספונסיבי
- **Shadcn/UI** - רכיבי UI מותאמים אישית
- **React Query** - ניהול מצב ונתונים

### Backend
- **Supabase** - מסד נתונים PostgreSQL
- **Row Level Security** - אבטחת מידע
- **Edge Functions** - לוגיקה עסקית
- **Real-time** - עדכונים בזמן אמת

### אינטגרציות
- **Google Calendar** - סנכרון אירועים
- **WhatsApp API** - שליחת הודעות
- **Google Maps** - מיקומים וניווט

## 🎨 תמיכה בעברית (RTL)

המערכת בנויה מהיסוד לתמיכה מלאה בעברית:

### הגדרות CSS בסיסיות
```css
html {
  direction: rtl;
  lang: he;
}

body {
  direction: rtl;
  text-align: right;
  font-family: system-ui, -apple-system, sans-serif;
}
```

### קלאסים מותאמים ל-RTL
```css
.rtl-flex {
  direction: rtl;
}

.space-x-reverse > :not([hidden]) ~ :not([hidden]) {
  --tw-space-x-reverse: 1;
}
```

### רכיבי UI
- כל הטפסים כוללים `dir="rtl"`
- תפריטים נפתחים מימין לשמאל
- הודפסת טקסט מימין לשמאל
- מיקום אייקונים מותאם לעברית

## 🚀 התחלה מהירה

### דרישות מערכת
- Node.js 18+
- npm או yarn
- חשבון Supabase

### התקנה
```bash
# שכפול הפרויקט
git clone <repository-url>
cd allforyou-system

# התקנת תלויות
npm install

# הגדרת משתני סביבה
cp .env.example .env.local

# הפעלת שרת הפיתוח
npm run dev
```

### הגדרת Supabase
1. צור פרויקט חדש ב-Supabase
2. העתק את ה-URL וה-API Key
3. הרץ את migrations:
```bash
supabase db reset
```
